Below are the CSS snipets for each of the animations included in this EPUB. They are presented as is from the original Animate.css files. However, the following animations from Animate.css were not included as they seemed repetitive for the purposes here: fadeInDownBig, fadeInLeftBig, fadeInRightBig, fadeInUpBig, fadeOutDownBig, fadeOutLeftBig, fadeOutRightBig, fadeOutUpBig, slideInDown, slideInLeft, slideInRight, slideInUp, slideOutDown, slideOutLeft, slideOutRight, slideOutUp.

The full CSS, including for those animations not included here, may be accessed at http://daneden.github.io/animate.css or by unzipping this EPUB and locating them in the styles folder.

Using these animations

  1. Copy the animate.css file into your EPUB file with the rest of your html/xhtml content docs, your images and any other stylesheets you may have. If there is not one already, I recommend creating a folder called styles, and adding it there.
  2. Add the following code to the head of your html document(s):
    <head>
  <link rel="stylesheet" href="styles/animate.css">
</head>
  3. Add the class animated to the element you want to animate. You may also want to include the class infinite for an infinite loop.
  4. Finally you need to add one of the following classes. For example, to animate a heading with a bounce that loops again and again:
    <h1 class="animated infinite bounce">BOUNCE!<h1>
    Or for a heading that zooms in only once:
    <h1 class="animated zoomIn">Zoom in once!<h1>
